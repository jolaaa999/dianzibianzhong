# 数字编钟击锤 — 使用说明

> 快速上手，5 分钟学会

---

## 一、拿到锤子第一步：配网

**锤子需要连 WiFi 才能和电脑通信。**

### 怎么判断要不要配网？

锤子插上 USB 通电，等 30 秒，看电脑 WiFi 列表：

- **出现了** `BianzongHammer-XXXXXX` → 需要配网
- **没出现** → 已连上 WiFi，跳到"启动软件"

### 配网步骤（3 分钟）

```
① 电脑连锤子热点：BianzongHammer-XXXXXX
   密码：12345678

② 浏览器打开：192.168.4.1

③ 点「刷新」，选你的 WiFi，填密码

④ 点「开始配网」，等显示「配网成功」

⑤ 电脑连回你的 WiFi
```

> 注意：锤子只支持 2.4GHz WiFi

## 二、启动软件

### 第一次运行

```powershell
# 进目录
cd bianzhong_ninja_copy

# 装依赖
flutter pub get

# 启动
flutter run -d windows
```

### 防火墙放行（必须！）

**管理员身份**打开 PowerShell，执行：

```powershell
netsh advfirewall firewall add rule name="BianzongHammer UDP 3333" dir=in action=allow protocol=UDP localport=3333
```

### 没锤子也能试

```powershell
flutter run -d windows --dart-define=MOCK=swinger
```

- **蓝色光标** = 左手锤（先连接的）
- **黄色光标** = 右手锤（后连接的）
- 每个钟可以敲**正面**（橙色区）或**侧面**（蓝色区）

## 四、怎么演奏

### 移动光标

```
锤子左右转  →  光标左右移
锤子前倾    →  光标下移
锤子后仰    →  光标上移
```

### 敲钟

**把光标移到钟上，然后向前推锤子** → 钟就响了。

四种方式都能敲响：前推、猛挥、快转、用力挥。通常"前推"最好用。

> 光标滑过钟时会记住位置，前推时即使光标被甩出去也能敲中。

### 连续敲

敲完一个马上可以敲下一个，两个钟之间几乎没有间隔。

---

## 五、跟奏模式（跟着谱子弹）

点击 **「智能跟奏」** → 选歌 → 倒计时 3、2、1

- 该敲的钟会**闪烁高亮**
- 敲对了 → 加分
- 错过了 → 扣分
- 弹完 → 看成绩

---

## 六、忍者模式（好玩的）

点击 **「水果忍者」** → 快速挥锤子 → 屏幕上留残影 → 残影划过钟就是斩击。

再点一下退出。

---

## 七、两个锤子一起用

两个锤子分别配好网，插上电就行。

App 自动分左右：
- 先上电的 = 左手（蓝）
- 后上电的 = 右手（黄）

## 八、设置（点 ⚙️）

| 功能 | 干嘛的 |
|------|--------|
| 音量 | 调大小 |
| 灵敏度 | 光标移动快慢 |
| 重启 UDP 监听 | 连不上时点这个 |
| 重置光标 | 光标回中间 |
| Mock 数据源 | 没锤子时模拟锤子 |

---

## 九、出问题了？

| 问题 | 解决 |
|------|------|
| 收不到锤子 | ① 防火墙放行了吗？② 同一个 WiFi 吗？③ 设置里重启 UDP |
| 光标不动 | 锤子串口有日志吗？BNO085 线松了？ |
| 敲不响 | 光标移到钟上再前推 |
| 方向反了 | 正常，改配置文件即可 |
| 光标乱跳 | 设置里点「重置光标」 |
